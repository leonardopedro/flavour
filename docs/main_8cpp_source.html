32 /** \mainpage Introduction
33 * The program produces figures presenting 68%, 95% and 99% CL allowed regions in parameter space.
34 * To wit, we represent regions where the specific BGL model is able to fit the imposed experimental information at least as well as the corresponding goodness levels.
35 * Some comments are in order.
36 * This procedure corresponds to the profile likelihood method.
37 * In brief, for a model with parameters \f$\vec p\f$, we compute the predictions for the considered set of observables \f$\vec O_{\mathrm{Th}}(\vec p)\f$.
38 * Then, using the experimental information \f$\vec O_{\mathrm{Exp}}\f$ available for those observables, we build a likelihood function \f$\mathcal L(\vec O_{\mathrm{Exp}}|\vec O_{\mathrm{Th}}(\vec p))\f$
39 * which gives the probability of obtaining the experimental results \f$\vec O_{\mathrm{\mathrm{Exp}}}\f$ assuming that the model is correct.
40 * The likelihood function \f$\mathcal L(\vec O_{\mathrm{Exp}}|\vec O_{\mathrm{Th}}(\vec p))\f$
41 * encodes all the information on how the model is able to reproduce the observed data all over parameter space.
42 * Nevertheless, the knowledge of \f$\mathcal L(\vec O_{\mathrm{Exp}}|\vec O_{\mathrm{Th}}(\vec p))\f$ in a multidimensional parameter space
43 * can be hardly represented and one is led to the problem of reducing that information to one or two-dimensional subspaces.
44 * In the profile likelihood method, for each point in the chosen subspace, the highest likelihood over the complementary, marginalized space, is retained. Let us clarify that likelihood
45 * -- or chi-squared \f$\chi^2\equiv -2\log \mathcal L\f$ --
46 * profiles and derived regions such as the ones we represent, are thus insensitive to the size of the space over which one marginalizes;
47 * this would not be the case in a Bayesian analysis, where an integration over the marginalized space is involved. The profile likelihood method seems adequate to our purpose,
48 * which is none other than exploring where in parameter space are the different BGL models able to satisfy experimental constraints,
49 * without weighting in eventual fine tunings of the models or parameter space volumes.
50 * For the numerical computations the libraries GiNaC and ROOT are used. *
51 */
